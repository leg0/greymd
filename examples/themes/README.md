# Themes

CSS themes for greymd. Each theme overrides page colors and syntax highlighting.

## Available Themes

| Theme | Style |
|-------|-------|
| `default` | Built-in colors (starting point for customization) |
| `catppuccin-latte` | Light, warm pastels |
| `catppuccin-frappe` | Muted dark |
| `catppuccin-macchiato` | Gentle dark |
| `catppuccin-mocha` | Rich dark |
| `tokyo-night` | Deep blue-black dark |

## Usage

Copy a theme's `css` file to `~/.config/greymd/css`:

```sh
cp examples/themes/catppuccin-mocha/css ~/.config/greymd/css
```

Changes take effect on the next page load — no restart needed.
